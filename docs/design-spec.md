## 一、视觉设计规范

### 色彩系统
```
背景层    #080c18  (深空黑)
卡片层    rgba(255,255,255,0.04) + border: 1px solid rgba(255,255,255,0.08)  (玻璃态)

危险/告警 #ff4757  (红)
文字主色  #e8f4ff
文字次色  #6b8aad
```

### 视觉关键词
- **Glassmorphism 卡片**：毛玻璃半透明 + 细描边
- **流光曲线**：实时折线用渐变拖尾
- **发光节点**：设备拓扑中反应釜节点脉冲发光，颜色编码状态
- **数字跳动**：首页 KPI 数字入场动画
- **粒子背景**：低密度流动粒子（Canvas）

### 字体
```
数据数字  JetBrains Mono（等宽）
中文标题  系统默认黑体
正文      Inter / -apple-system
```

## 二、技术选型

| 层级 | 选型 | 用途 |
|------|------|------|
| 框架 | React 18 + TypeScript + Vite | 核心 |
| UI组件 | Ant Design 5（暗色 token 定制） | 表格/表单/弹窗 |
| 图表 | ECharts 5 | 折线/散点/热力图/雷达图/箱线图 |
| 拖拽流程 | React Flow | 实验流程编排画布 |
| 分子结构 | SmilesDrawer.js | SMILES → 2D 分子结构图 |
| 动画 | Framer Motion | 页面切换/数字跳动/告警动效 |
| 样式 | Tailwind CSS + CSS 变量 | 暗色主题统一管理 |
| Mock | 纯 useState + setInterval | 模拟实时数据流 |
| 路由 | React Router 6 | 多页面导航 |

## 四、各页面详细设计

### P1 — 首页大屏
**定位**：进入系统的第一眼，冲击力最强

**布局（4区块）**：
```
┌─────────────────────────────────────────────────┐
│  LOGO  CatalystOS          [时间] [告警] [用户]  │
├──────────┬──────────────────────────┬────────────┤
│          │                          │            │
│  今日    │   实验室设备拓扑动画      │  实时告警  │
│  工单    │   (SVG 32釜 + AGV路径)   │  列表      │
│  进度    │                          │  + AI推荐  │
│  甘特条  │                          │  预览      │
├──────────┴──────────────────────────┴────────────┤
│  [KPI卡] 今日批次 | 运行中反应釜 | AI准确率 | 运行时长  │
├─────────────────────────────────────────────────┤
│  活性分布趋势 | PDI 均值周趋势 | 科研产出累计统计   │
└─────────────────────────────────────────────────┘
```

**关键动效**：
- 反应釜节点：运行中→青蓝脉冲光晕；告警→橙色闪烁；离线→灰色
- AGV 小车：白色圆点沿路径匀速流动
- KPI 数字：页面载入时从 0 滚动到目标值（CountUp）
- 拓扑连线：数据流向用流光动画

**Mock 数据**：
```
今日批次: 47  |  运行中反应釜: 28/32  |  AI推荐准确率: 87.3%
系统连续运行: 312h 47m  |  本月节省实验次数: 183次
累计发现高活性配方: 24个  |  较传统方法效率提升: 8.6x
```

### P2 — 反应过程监控
**布局**：
```
┌──────────────────────────┬─────────────────────┐
│  实时色谱图               │  反应趋势图          │
│  (Online GC 多峰动态)     │  (转化率 vs 取样批次)│
├──────────────────────────┴─────────────────────┤
│  [反应状态]                │  [反应历史与产物标定]│
│  状态: 已完成              │  取样1/2/3/4 切换    │
│  目标产物浓度: 86%         │  序号|物质标识|RT/min│
│  反应程度: 过高 ⚠          │  |Ara%|类别|操作     │
└────────────────────────────────────────────────┘
```

**关键交互**：
- 实时色谱图：setInterval 每 800ms 更新一帧，曲线平滑流动
- 聚合放热曲线：温度呈 S 型上升（模拟 Cossee-Arlman 放热）
- 告警触发：压力突破阈值→红色标注+联锁处置弹窗

## 五、Mock 数据关键设计

### 实时数据模拟（setInterval）
```typescript
// 温度曲线模拟：聚合放热 S 型上升
temperature = baseTemp + amplitude * (1 / (1 + e^(-k*(t-t0)))) + noise()

// 在线GC 色谱图：多峰高斯叠加
peak(x) = height * exp(-((x-center)^2 / (2*width^2))) + noise(0.5)

// 刷新频率
温度/压力曲线: 1000ms
色谱图:        800ms
设备状态:      3000ms
AGV位置:       500ms
```

## 六、Demo 演示剧本（建议顺序，约8分钟）

```
1. 首页大屏        ← 第一印象（30秒）拓扑动画 + KPI数字跳动
2. AI方案推荐      ← 解释核心价值（2分钟）分子结构+SHAP+贝叶斯图
   一键同意 → 生成工单
3. 工单看板        ← 工单出现在"调度中"（30秒）
4. 反应过程监控    ← 实验实时进行（1分钟）触发告警演示+联锁
5. 谱图分析        ← 实验完成，谱图自动解析（1分钟）
6. 智能排产        ← 高通量效率展示（30秒）
7. 催化剂数据库    ← 结构可视化（30秒）
8. 数据大屏        ← 历史积累价值（30秒）
9. 实验报告        ← 合规审计（30秒）
```
